# JDBCmini_project
This is a basic Java JDBC project that demonstrates how to build a simple database-driven desktop application using Java Swing for the user interface and JDBC for database connectivity. The project is structured into multiple classes to separate functionality and improve readability.

The application includes several modules such as login, insert, delete, search, query execution, and data display. Each feature is implemented in its own dedicated class (e.g., LoginPage, InsertPage, SearchPage, DisplayPage, etc.), while the HomePage acts as the main navigation interface, allowing users to access different operations through buttons.

The project showcases:

Establishing database connections using JDBC
Executing SQL queries (INSERT, DELETE, SELECT)
Building a GUI using Java Swing components like JFrame, JPanel, and JButton
Handling user actions with event listeners
Organizing code into modular, maintainable classes

This project is ideal for beginners who want to understand the fundamentals of integrating Java applications with databases and creating simple GUI-based tools.
